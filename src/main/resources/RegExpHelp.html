Regular expressions are a powerful way to subset files. They are not only used here in the BigDataProcessor2 but are widely used in many software packages.

Useful examples

.* Selects all files
.*.tif$ Only open files ending with .tif
.*.h5$ Only open files ending with .h5
^((?!_All).)*.h5$ Luxendo: Only open files ending with .h5, but not containing the text "All".
